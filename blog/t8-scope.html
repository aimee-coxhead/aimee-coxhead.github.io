Scope

What is 'scope', and how does it work in JavaScript?

The scope is what variables you have access to when a piece of code is running. In JavaScript there is two types of scope; local and global. When variables are created inside a function they are part of the local scope and can only be accessed in that function. This means that you could create a variable with the same name inside another function because the variables can only be used inside their own functions and can not be accessed in other functions. When variables are created outside of a function they are part of the global scope and can be assessed from anywhere, including inside functions.
